## 🎯 Objectives
- **Primary Objective**: Design and build a lightweight, high-performance electric longboard optimized for urban commuting.

  - 🌍 Use eco-friendly materials and components.
  - 📱 Integrate a user-friendly app and remote for diagnostics and control.

## 🤝 User Interaction

### a. 📦 Unboxing
- The user unpacks a minimalistic box containing the board, charger, and remote. The packaging is eco-friendly and made from recyclable materials. A quick start guide is included.

### b. 🔧 Setting Up
- Users download the mobile application, connect it to the board via Bluetooth, and configure ride settings like speed mode and brake sensitivity.

### c. 🚀 Using
- Press the power button, select a speed mode via the app, and use the remote to throttle and brake. LED indicators on the board display battery status.

### d. 🛠️ Troubleshooting
- The app displays error codes, such as low battery or motor faults. FAQs within the app help users troubleshoot common issues, such as resetting the ECU.

### e. 🔩 Servicing
- Users can view service logs in the app, including recommended maintenance schedules:
  - Tire Replacement: Every 500 miles.
  - Battery Check: After 200 charges.

---

## 📱 User Interface

### App UI

## ⚙️ Parameters

- **Range**: 20 miles per charge.

### b. 📏 Dimensions
- **Deck Length**: 38 inches.
- **Width**: 9 inches.

### c. ⚖️ Weight
- **Total Weight**: 16 pounds.

### d. 🌧️ Protection

### e. 🔋 Core Functions
- Speed mode control: Eco, Normal, Sport.
- Regenerative braking system.
- Battery monitoring and alerts.
